# WireGuard Bot — Управление VPN через Telegram Bot

Простая установка VPN-сервера и Telegram-бота для управления клиентами — всё одной командой.

## ✨ Возможности
- ➕ **Добавление клиентов** — выбор срока доступа (24 часа / 1-12 месяцев / постоянный)
- ❌ **Удаление клиентов** — мгновенное отключение любого пользователя
- 📋 **Список клиентов** — список всех подключённых и их IP
- 📊 **Статистика** — информация о состоянии, активных подключениях и о траффике.
- 🔐 **Белый список** — доступ к боту только у указанных Telegram ID
- ⚡ **Systemd** — автозапуск сервисов при старте системы

## 🚀 Быстрый запуск

```bash
# 1. Клонируем репозиторий
git clone https://github.com/SergeyIlins/wireguard-bot.git
cd wireguard-bot

# 2. Настраиваем окружение
cp .env.example .env
nano .env   # Укажите TELEGRAM_BOT_TOKEN и ADMIN_IDS

# 3. Запускаем установку
chmod +x install.sh
sudo ./install.sh

Важно: Установка работает только на серверах за пределами страны блокировки.
